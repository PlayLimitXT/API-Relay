# 更新日志

## v1.2.0 (2026-04-16)

### 新增功能
- **国际化 (i18n) 支持**：管理面板支持中文/英文切换，所有界面文本完整翻译
- **OpenAI SDK 集成**：添加 `openai>=1.12.0` 依赖，支持 兼容 OpenAI API 的服务
- **环境变量配置示例**：新增 `.env.example` 环境变量配置示例文件
- **修复隐私过滤功能导致的错误**: 修复了开启隐私过滤可能导致大模型API无法正常使用的问题
- **修复数据导入导出功能**: 修复了无法正常导入数据的问题
- **修复数据统计功能的已知问题**: 修复多处错误
- **新增获取源 API 模型功能**: 一键获取源API支持的模型ID

### 修复
- **模型保存功能修复**：修复编辑源 API 时模型列表未正确保存的 bug（`updateSource` 函数错误使用 `editingSourceModels` 而非 `sourceModelsList`）
- **获取源 API 模型功能**：修复 `Not Found` 错误，确保模型列表正确获取
- **修复页面刷新 session 丢失问题**：修复了管理员会话在页面刷新后无法保持的问题
- **修复会话验证时序问题**：确保 sessionToken 在 API 请求前完成验证
- **修复初始化时序冲突**：解决了页面加载时 `loadPrivacySettings()` 在会话验证前发起请求导致 401 的问题
- **修复 `applyLanguage()` 会话检查缺失**：添加未登录状态检查，避免无效的 API 请求
- **前端代码语法修复**：修复 sed 操作导致的 `no_data` 字符串语法错误

### 优化
- 管理面板 UI 改进，所有按钮、表单、提示文本完整国际化
- 语言选择器高亮显示当前语言
- 改进会话持久化机制
- 优化初始化流程，避免竞态条件
- 增强错误处理和日志记录

## v1.1.0 (2026-04-10)

### 新增功能
- **隐私过滤**：支持过滤源 API 返回的敏感信息（元数据、使用详情、提供商信息等）
- **自定义错误消息**：可配置统一的错误响应格式
- **流式响应隐私过滤**：SSE 流式请求也支持隐私过滤

### 优化
- 更详尽的统计
- 改进输入输出比显示格式（`x.xxx:1` 或 `1:x.xxx`）
- 改进管理面板 UI/UX
- 优化配置保存逻辑

### 修复
- 修复多个显示和统计 bug
